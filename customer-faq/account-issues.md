# 客户账户问题 FAQ

## 标签
`#faq` `#account` `#login`

## 常见问题

### 1. 无法登录账户

**问题描述**：输入正确的邮箱和密码后仍然无法登录

**可能原因**：
- 账户被禁用
- 浏览器缓存问题
- 网络连接问题

**解决方案**：
1. 清除浏览器缓存和 Cookie
2. 尝试使用无痕模式
3. 检查网络连接
4. 如果仍然无法解决，请联系 support@storellet.com

### 2. 忘记密码

**问题描述**：忘记登录密码

**解决方案**：
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收密码重置邮件（可能在垃圾邮件文件夹）
4. 点击重置链接设置新密码

**注意**：重置链接 24 小时内有效

### 3. 账户被锁定

**问题描述**：多次输入错误密码后账户被锁定

**解决方案**：
- 等待 30 分钟后自动解锁
- 或联系管理员手动解锁

**预防措施**：
- 启用双因素认证（2FA）
- 使用密码管理器
- 定期更换密码

### 4. 邮箱地址变更

**问题描述**：需要更改注册邮箱

**操作步骤**：
1. 登录后进入"账户设置"
2. 点击"更改邮箱"
3. 输入新邮箱地址
4. 验证新邮箱（会发送验证邮件）
5. 验证完成后邮箱更新成功

### 5. 删除账户

**问题描述**：用户希望删除自己的账户

**操作步骤**：
1. 登录后进入"账户设置"
2. 滚动到页面底部
3. 点击"删除账户"
4. 阅读警告信息并确认
5. 输入密码确认删除

**注意**：
- 删除操作不可逆
- 所有数据将被永久删除
- 建议先导出重要数据

### 6. 权限不足

**问题描述**：访问某些功能时提示"权限不足"

**可能原因**：
- 账户角色权限不足
- 功能需要特定权限
- 账户已被限制

**解决方案**：
1. 联系管理员确认账户角色
2. 申请相应的权限
3. 或请求有权限的同事协助
