## 🔧 如何修改配置

### 方法 1：直接编辑 JSON 文件

1. 打开 `chatbot-config.json`
2. 修改内容
3. 提交到 GitHub

**添加新问题示例**：
```json
{
  "id": 9,
  "question": "如何批量调整积分？",
  "category": "营销功能",
  "priority": "medium"
}
```

**删除问题示例**：
- 从 `quickQuestions` 数组中删除对应的对象
- 重新排序 ID（可选）

**修改问题示例**：
```json
{
  "id": 1,
  "question": "如何添加新员工？",  // 修改这里
  "category": "用户管理",
  "priority": "high"
}
```

### 方法 2：在 Chatbot 中管理（推荐）

**步骤**：
1. 打开 Chatbot 页面
2. 点击刷新按钮获取最新配置
3. 测试新的快速问题

## 📝 修改步骤

1. **编辑配置文件**
   ```bash
   # 在 GitHub 上编辑
   # 或者本地编辑后 push
   vim chatbot-config.json
   ```

2. **提交到 GitHub**
   ```bash
   git add chatbot-config.json
   git commit -m "update: 修改快速问题配置"
   git push origin main
   ```

3. **在 Chatbot 中刷新**
   - 点击 "🔄 刷新知识库"
   - 新配置立即生效
