## 🔧 如何修改配置

### 方法 1：直接編輯 JSON 文件

1. 打開 `chatbot-config.json`
2. 修改內容
3. 提交到 GitHub

**添加新問題示例**：
```json
{
  "id": 9,
  "question": "如何批量調整積分？",
  "category": "營銷功能",
  "priority": "medium"
}
```

**刪除問題示例**：
- 從 `quickQuestions` 數組中刪除對應的對象
- 重新排序 ID（可選）

**修改問題示例**：
```json
{
  "id": 1,
  "question": "如何添加新員工？",  // 修改這裡
  "category": "用戶管理",
  "priority": "high"
}
```

### 方法 2：在 Chatbot 中管理（推薦）

**步驟**：
1. 打開 Chatbot 頁面
2. 點擊刷新按鈕獲取最新配置
3. 測試新的快速問題

## 📝 修改步驟

1. **編輯配置文件**
   ```bash
   # 在 GitHub 上編輯
   # 或者本地編輯後 push
   vim chatbot-config.json
   ```

2. **提交到 GitHub**
   ```bash
   git add chatbot-config.json
   git commit -m "update: 修改快速問題配置"
   git push origin main
   ```

3. **在 Chatbot 中刷新**
   - 點擊 "🔄 刷新知識庫"
   - 新配置立即生效
