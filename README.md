# CMS Knowledge Base

## 📚 概述

这是 Storellet CMS 的官方知识库，用于支持内部团队和客户服务。

## 🎯 用途

- **内部培训**：新员工学习 CMS 操作
- **客户支持**：快速查找客户问题答案
- **故障排除**：解决常见技术问题
- **API 参考**：开发集成指南
- **智能问答**：通过 Chatbot 获取即时答案

## ✨ 已完成模块

### 操作指南（9个）

#### 基础操作
- **登录指南** - 登录步骤、密码管理、错误处理
- **密码重置指南** - 3种重置方法、安全最佳实践

#### 用户与权限管理
- **员工管理** - 员工CRUD、权限设置、密码管理
- **会员管理** - 会员信息、积分、优惠券、印花卡管理

#### 数据分析与报表
- **仪表板使用** - 数据查看、筛选、下钻、导出
- **报表与分析** - 概览报表、自定义报表、数据导出

#### 系统配置
- **应用配置管理** - 前端页面、图片、标签、地区、POS密钥、PMS集成

#### 营销与组织
- **推送管理** - 优惠券推送、积分批量调整、推送通知、新闻推送
- **公司品牌管理** - 公司、组/品牌、店铺、特色项目管理

### 代码参考（1个）

- **AdminAction** - 400+行详细代码参考，15个API端点分析

## 🔄 如何更新

1. 在相应文件夹创建新文档
2. 使用清晰的 Markdown 格式
3. 添加适当的标签和关键词
4. 提交 PR 进行审查

## 🔍 搜索技巧

- 使用文件名中的关键词搜索
- 查看每个文档的标签部分
- Chrome Extension 会自动匹配相关内容
