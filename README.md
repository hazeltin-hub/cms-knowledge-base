# CMS Knowledge Base

## 📚 概述

这是 Storellet CMS 的官方知识库，用于支持内部团队和客户服务。

## 🎯 用途

- **内部培训**：新员工学习 CMS 操作
- **客户支持**：快速查找客户问题答案
- **故障排除**：解决常见技术问题
- **API 参考**：开发集成指南

## 📁 结构

```
cms-knowledge-base/
├── cms-operations/          # CMS 操作指南
├── customer-faq/            # 客户常见问题
├── troubleshooting/         # 故障排除
└── api-reference/           # API 参考文档
```

## 🔄 如何更新

1. 在相应文件夹创建新文档
2. 使用清晰的 Markdown 格式
3. 添加适当的标签和关键词
4. 提交 PR 进行审查

## 🔍 搜索技巧

- 使用文件名中的关键词搜索
- 查看每个文档的标签部分
- Chrome Extension 会自动匹配相关内容
