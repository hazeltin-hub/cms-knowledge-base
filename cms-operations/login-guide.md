# CMS 登錄指南

## 標簽
#user-guide #login #beginner

## 概述

本文檔介紹如何登錄 Storellet CMS 系統，包括首次登錄、忘記密碼處理和常見登錄問題。

---

## 🔐 登錄步驟

### 方法一：使用用戶名和密碼登錄

1. **訪問登錄頁面**
   ```
   URL: https://your-domain.com/cms/login
   ```

2. **輸入登錄信息**
   - **用戶名**：輸入您的用戶名
   - **密碼**：輸入您的密碼

3. **點擊"登錄"按鈕**

4. **登錄成功**
   - 系統會跳轉到儀表板
   - 如果是首次登錄，可能需要修改密碼

### 方法二：使用"忘記密碼"功能

如果忘記密碼，可以通過用戶名重置：

1. **訪問密碼重置頁面**
   ```
   URL: /cms/changePasswordWithUsername
   ```

2. **填寫信息**
   - **用戶名**：輸入您的用戶名
   - **舊密碼**：留空（如果忘記）
   - **新密碼**：輸入新密碼
   - **確認密碼**：再次輸入新密碼

3. **提交**
   - 點擊"提交"按鈕
   - 系統會驗證用戶名並更新密碼

---

## 🔑 密碼要求

### 密碼策略

- **最小長度**：至少 6 個字符
- **復雜度**：建議包含字母、數字和特殊字符
- **密碼過期**：如果密碼超過 365 天未修改，系統可能要求修改

### 修改密碼時機

系統會在以下情況提示修改密碼：
- 首次登錄
- 密碼超過 365 天未修改
- 管理員強制要求修改密碼

---

## 🔑 已登錄用戶修改密碼

如果您已經登錄，想修改當前密碼：

### 步驟 1：訪問密碼修改頁面

1. 在瀏覽器中訪問：
   ```
   /cms/changePassword
   ```

2. 系統會自動驗證您的登錄狀態

### 步驟 2：填寫表單

- **舊密碼**：輸入當前使用的密碼
- **新密碼**：輸入新密碼
- **確認密碼**：再次輸入新密碼

### 步驟 3：提交

- 點擊"提交"按鈕
- 系統會驗證舊密碼是否正確
- 密碼修改成功後會自動更新會話

### 注意事項

⚠️ **重要**：
- 新密碼必須輸入兩次以避免拼寫錯誤
- 如果舊密碼輸入錯誤超過 3 次，賬戶可能被臨時鎖定
- 密碼修改後，系統會記錄修改時間

## ❌ 登錄失敗處理

### 常見錯誤

#### 錯誤 1：用戶名或密碼錯誤

**症狀**：
```
用戶名或密碼錯誤
```

**解決方案**：
1. 檢查用戶名拼寫是否正確
2. 確認 Caps Lock（大寫鎖定）未開啟
3. 點擊"忘記密碼"重置密碼
4. 聯系管理員確認賬戶狀態

#### 錯誤 2：賬戶被禁用

**症狀**：
```
您的賬戶已被禁用，請聯系管理員
```

**解決方案**：
1. 聯系系統管理員
2. 確認賬戶狀態
3. 等待管理員重新啟用賬戶

#### 錯誤 3：密碼過期

**症狀**：
```
您的密碼已過期，請修改密碼
```

**解決方案**：
1. 使用"忘記密碼"功能重置密碼
2. 或聯系管理員協助重置

#### 錯誤 4：會話超時

**症狀**：
```
會話已過期，請重新登錄
```

**解決方案**：
1. 重新輸入用戶名和密碼
2. 系統會創建新的會話

## 🛡️ 安全建議

### 密碼安全

- ✅ **不要使用簡單密碼**：如 123456、password 等
- ✅ **定期更換密碼**：建議每 3-6 個月更換一次
- ✅ **不要重復使用密碼**：避免與其他網站使用相同密碼
- ✅ **不要分享密碼**：不要告訴任何人您的密碼

### 登錄安全

- ✅ **退出登錄**：使用完成後記得點擊"退出"
- ✅ **不要在公共電腦上選擇"記住我"**
- ✅ **檢查網址**：確認登錄頁面網址正確
- ✅ **警惕釣魚網站**：只訪問官方 CMS 登錄頁面

## 🔧 故障排除

### 問題：登錄後自動退出

**可能原因**：
- 瀏覽器 Cookie 被禁用
- 瀏覽器設置問題
- 系統時間不正確

**解決方案**：
1. 檢查瀏覽器是否啟用 Cookie
2. 清除瀏覽器緩存和 Cookie
3. 嘗試使用其他瀏覽器（Chrome、Firefox、Safari）

### 問題：密碼重置郵件未收到

**可能原因**：
- 郵件地址錯誤
- 郵件被歸類為垃圾郵件
- 郵件服務器延遲

**解決方案**：
1. 檢查垃圾郵件文件夾
2. 確認郵箱地址是否正確
3. 等待 5-10 分鐘再檢查
4. 聯系管理員手動重置

### 問題：無法訪問登錄頁面

**可能原因**：
- 網絡連接問題
- 服務器維護
- 防火牆阻止

**解決方案**：
1. 檢查網絡連接
2. 嘗試訪問其他網站確認網絡正常
3. 聯系 IT 部門確認服務器狀態
4. 嘗試從其他網絡或設備訪問
