# CMS 登录指南

## 标签
#user-guide #login #beginner

## 概述

本文档介绍如何登录 Storellet CMS 系统，包括首次登录、忘记密码处理和常见登录问题。

---

## 🔐 登录步骤

### 方法一：使用用户名和密码登录

1. **访问登录页面**
   ```
   URL: https://your-domain.com/cms/login
   ```

2. **输入登录信息**
   - **用户名**：输入您的用户名
   - **密码**：输入您的密码

3. **点击"登录"按钮**

4. **登录成功**
   - 系统会跳转到仪表板
   - 如果是首次登录，可能需要修改密码

### 方法二：使用"忘记密码"功能

如果忘记密码，可以通过用户名重置：

1. **访问密码重置页面**
   ```
   URL: /cms/changePasswordWithUsername
   ```

2. **填写信息**
   - **用户名**：输入您的用户名
   - **旧密码**：留空（如果忘记）
   - **新密码**：输入新密码
   - **确认密码**：再次输入新密码

3. **提交**
   - 点击"提交"按钮
   - 系统会验证用户名并更新密码

---

## 🔑 密码要求

### 密码策略

- **最小长度**：至少 6 个字符
- **复杂度**：建议包含字母、数字和特殊字符
- **密码过期**：如果密码超过 365 天未修改，系统可能要求修改

### 修改密码时机

系统会在以下情况提示修改密码：
- 首次登录
- 密码超过 365 天未修改
- 管理员强制要求修改密码

---

## 🔑 已登录用户修改密码

如果您已经登录，想修改当前密码：

### 步骤 1：访问密码修改页面

1. 在浏览器中访问：
   ```
   /cms/changePassword
   ```

2. 系统会自动验证您的登录状态

### 步骤 2：填写表单

- **旧密码**：输入当前使用的密码
- **新密码**：输入新密码
- **确认密码**：再次输入新密码

### 步骤 3：提交

- 点击"提交"按钮
- 系统会验证旧密码是否正确
- 密码修改成功后会自动更新会话

### 注意事项

⚠️ **重要**：
- 新密码必须输入两次以避免拼写错误
- 如果旧密码输入错误超过 3 次，账户可能被临时锁定
- 密码修改后，系统会记录修改时间

## ❌ 登录失败处理

### 常见错误

#### 错误 1：用户名或密码错误

**症状**：
```
用户名或密码错误
```

**解决方案**：
1. 检查用户名拼写是否正确
2. 确认 Caps Lock（大写锁定）未开启
3. 点击"忘记密码"重置密码
4. 联系管理员确认账户状态

#### 错误 2：账户被禁用

**症状**：
```
您的账户已被禁用，请联系管理员
```

**解决方案**：
1. 联系系统管理员
2. 确认账户状态
3. 等待管理员重新启用账户

#### 错误 3：密码过期

**症状**：
```
您的密码已过期，请修改密码
```

**解决方案**：
1. 使用"忘记密码"功能重置密码
2. 或联系管理员协助重置

#### 错误 4：会话超时

**症状**：
```
会话已过期，请重新登录
```

**解决方案**：
1. 重新输入用户名和密码
2. 系统会创建新的会话

## 🛡️ 安全建议

### 密码安全

- ✅ **不要使用简单密码**：如 123456、password 等
- ✅ **定期更换密码**：建议每 3-6 个月更换一次
- ✅ **不要重复使用密码**：避免与其他网站使用相同密码
- ✅ **不要分享密码**：不要告诉任何人您的密码

### 登录安全

- ✅ **退出登录**：使用完成后记得点击"退出"
- ✅ **不要在公共电脑上选择"记住我"**
- ✅ **检查网址**：确认登录页面网址正确
- ✅ **警惕钓鱼网站**：只访问官方 CMS 登录页面

## 🔧 故障排除

### 问题：登录后自动退出

**可能原因**：
- 浏览器 Cookie 被禁用
- 浏览器设置问题
- 系统时间不正确

**解决方案**：
1. 检查浏览器是否启用 Cookie
2. 清除浏览器缓存和 Cookie
3. 尝试使用其他浏览器（Chrome、Firefox、Safari）

### 问题：密码重置邮件未收到

**可能原因**：
- 邮件地址错误
- 邮件被归类为垃圾邮件
- 邮件服务器延迟

**解决方案**：
1. 检查垃圾邮件文件夹
2. 确认邮箱地址是否正确
3. 等待 5-10 分钟再检查
4. 联系管理员手动重置

### 问题：无法访问登录页面

**可能原因**：
- 网络连接问题
- 服务器维护
- 防火墙阻止

**解决方案**：
1. 检查网络连接
2. 尝试访问其他网站确认网络正常
3. 联系 IT 部门确认服务器状态
4. 尝试从其他网络或设备访问
