# CMS 会员管理指南

## 标签
#user-guide #membership #customer #intermediate

## 概述

本文档介绍如何在 Storellet CMS 中管理会员账户，包括会员信息查看、编辑、积分管理、优惠券管理和历史记录查询。

---

## 👥 访问会员管理

### 步骤 1：进入会员管理页面

1. 登录 CMS 系统
2. 点击顶部导航的**"会员"**或**"Membership"**
3. 系统会跳转到会员管理首页
   ```
   URL: /cms/membership/index
   ```

### 步骤 2：选择品牌和店铺

在会员管理页面，需要先选择：
- **品牌 (Brand)**：选择要管理的品牌
- **店铺 (Shop)**：选择具体店铺（可选）

---

## 🔍 查找会员

### 方法 1：通过会员 ID 查找

1. 在会员管理页面
2. 输入**会员 ID**
3. 点击"查找"或"搜索"
4. 系统会显示该会员的详细信息

### 方法 2：通过手机号查找

1. 输入完整的手机号（包含国家代码）
   - 例如：`+852-12345678` 或 `85212345678`
2. 点击"搜索"
3. 系统会显示匹配的会员列表

**注意事项**：
- ⚠️ 手机号必须包含国家代码
- ⚠️ 格式：`+[国家代码][号码]` 或 `[国家代码][号码]`
- ✅ 支持的国家代码包括：852（香港）、86（中国）、886（台湾）等

### 方法 3：通过邮箱查找

1. 输入完整的邮箱地址
2. 点击"搜索"
3. 系统会显示匹配的会员

### 方法 4：通过会员卡/二维码查找

1. 点击**"二维码扫描"**或**"QR Code Inspector"**
2. 使用扫描设备扫描会员二维码
3. 系统会自动识别并显示会员信息
   ```
   URL: /cms/membership/qrCodeInspector
   ```

## 📝 查看会员详情

### 会员信息页面组成

#### 1. 基本信息

#### 3. 统计信息

- **订单数量**：历史订单总数

- **印花卡**：进行中的印花卡

## ✏️ 编辑会员信息

### 修改个人资料

#### 步骤 1：打开编辑页面

1. 在会员详情页面
2. 点击**"编辑"**或**"Edit"**按钮
3. 系统会显示编辑表单

#### 步骤 2：修改信息

可以修改的信息包括：

**基本信息**：
- ✅ **显示名称**：会员显示的名称
- ✅ **姓名**：名和姓
- ✅ **性别**：选择性别
- ✅ **生日**：选择生日日期

**联系方式**：
- ✅ **邮箱**：更改邮箱地址
  - ⚠️ 新邮箱不能与其他会员重复
  - ⚠️ 格式必须有效（如 user@example.com）
- ✅ **手机号**：更改手机号码
  - ⚠️ 必须包含国家代码
  - ⚠️ 新号码不能与其他会员重复
- ✅ **其他手机号**：添加备用手机号（可多个）

**社交媒体**：
- ✅ **Facebook ID**：关联 Facebook 账户
- ✅ **其他社交媒体**：根据品牌设置添加

#### 步骤 3：保存更改

1. 点击**"保存"**或**"Save"**按钮
2. 系统会验证所有输入
3. 保存成功后会显示确认消息
   ```
   URL: /cms/membership/updateProfile
   ```

#### 注意事项

**邮箱验证**：
- ⚠️ 修改邮箱后，系统可能会要求重新验证邮箱
- ⚠️ 会员需要点击验证链接完成验证

**手机号格式**：
- ⚠️ 必须包含有效的国家代码
- ⚠️ 例如：`+852-12345678`、`86-13812345678`

**重复检查**：
- ⚠️ 邮箱和手机号不能与现有会员重复
- ⚠️ 如果重复，系统会显示错误信息

## 💰 积分管理

### 查看积分详情

#### 访问积分页面

在会员详情页面：
1. 找到**"积分"**或**"Points"**部分
2. 可以查看：
   - **当前积分**：可用积分余额
   - **过期积分**：即将过期的积分
   - **积分历史**：积分变动记录

### 添加积分

#### 方法 1：手动添加积分

1. 点击**"添加积分"**或**"Add Points"**
2. 输入信息：
   - **积分数量**：正数表示添加
   - **原因**：添加原因（如活动奖励、补偿等）
   - **过期日期**：积分过期时间（可选）
3. 点击**"确认"**
   ```
   URL: /cms/membership/addGroupPoint
   ```

#### 方法 2：通过交易添加积分

1. 点击**"交易转积分"**或**"Transaction to Points"**
2. 输入交易 ID（Transaction ID）
3. 选择会员 ID
4. 系统会显示：
   - 交易金额
   - 应得积分
   - 今日已添加积分次数
   - 本周已添加积分次数
5. 确认后添加积分
   ```
   URL: /cms/membership/transaction2Member
   ```

**注意事项**：
- ⚠️ 交易不能是已撤销的
- ⚠️ 交易不能已经添加过积分
- ⚠️ 需要检查每日/每周积分添加限制

### 修改积分过期日期

1. 在积分详情页面
2. 找到**"过期日期"**
3. 点击**"编辑"**
4. 选择新的过期日期
5. 点击**"保存"**
   ```
   URL: /cms/membership/updateUserPointExpiryDate
   ```

**用途**：
- 延长会员积分有效期
- 作为特殊奖励延长积分

## 🎫 优惠券管理

### 查看会员优惠券

#### 访问优惠券页面

在会员详情页面：
1. 找到**"优惠券"**或**"Coupons"**部分
2. 可以查看：
   - **可用优惠券**：未使用且未过期
   - **已使用优惠券**：已经使用过的
   - **已过期优惠券**：已过期的
   ```
   URL: /cms/membership/{userId}/group/{groupId}/coupons
   ```

### 添加优惠券

#### 步骤 1：选择优惠券

1. 点击**"添加优惠券"**或**"Add Coupon"**
2. 系统会显示可用的优惠券列表
   ```
   URL: /cms/membership/availableCoupons
   ```
3. 选择要添加的优惠券类型

#### 步骤 2：配置优惠券

输入以下信息：
- **优惠券类型**：选择优惠券
- **数量**：发放数量
- **过期日期**：设置过期时间
- **备注**：添加原因（可选）

#### 步骤 3：确认发放

1. 点击**"发放"**
2. 系统会确认发放
   ```
   URL: /cms/membership/addFavour
   ```
3. 会员会收到通知（根据品牌设置）

### 使用优惠券（手动）

**场景**：会员无法自动使用优惠券时，客服可以手动操作

1. 输入优惠券序列号（Serial）
2. 点击**"使用"**
3. 系统会验证：
   - 优惠券是否属于该会员
   - 优惠券是否在有效期内
   - 优惠券是否已使用
4. 确认使用
   ```
   URL: /cms/membership/useUserFavour/{serial}
   ```

### 作废优惠券

1. 找到要作废的优惠券
2. 点击**"作废"**或**"Expire"**
3. 输入作废原因
4. 确认作废
   ```
   URL: /cms/membership/expireUserFavour/{serial}
   ```

**用途**：
- 优惠券出现错误需要作废
- 防止错误使用

### 修改优惠券过期日期

1. 在优惠券列表中
2. 点击**"编辑过期日期"**
3. 选择新的过期日期
4. 点击**"保存"**
   ```
   URL: /cms/membership/updateFavourExpiryDate
   ```

## 📜 查看历史记录

### 会员活动历史

#### 访问历史页面

1. 在会员详情页面
2. 点击**"历史记录"**或**"History"**
3. 系统会显示所有活动记录
   ```
   URL: /cms/membership/{userId}/group/{groupId}/history
   ```

#### 历史记录类型

显示的活动类型包括：

| 类型 | 说明 |
|------|------|
| **积分变动** | 获得积分、使用积分、积分过期 |
| **印花收集** | 获得印花、完成印花卡 |
| **优惠券** | 获得优惠券、使用优惠券、优惠券过期 |
| **等级变动** | 等级提升、等级降低 |
| **资料更新** | 修改个人资料、联系方式 |
| **交易记录** | 消费交易、退款 |

#### 筛选历史记录

**按时间筛选**：
- 今天
- 本周
- 本月
- 自定义日期范围

**按类型筛选**：
- 只看积分记录
- 只看优惠券记录
- 只看印花记录

**导出历史**：
- 点击**"导出"**
- 选择格式（Excel/CSV）
- 下载历史记录

### 查看历史总数

1. 在历史页面
2. 点击**"统计"**或**"Summary"**
3. 系统会显示：
   - 总积分变动
   - 总获得印花数
   - 总使用优惠券数
   - 总消费金额
   ```
   URL: /cms/membership/{userId}/group/{groupId}/getHistoryTotal
   ```

## 🔧 常见问题

### Q: 会员信息修改后不生效？

**A**：
1. 清除浏览器缓存并刷新页面
2. 确认修改已保存（检查成功消息）
3. 如果修改联系方式，会员可能需要重新登录

### Q: 添加积分失败？

**A**：
1. 检查会员是否属于正确的组
2. 确认积分数量是否正确（正数）
3. 检查是否超出每日/每周限制
4. 确认交易 ID 有效（如果是通过交易添加）

### Q: 无法找到会员？

**A**：
1. 确认搜索格式正确
2. 尝试使用会员 ID 而不是手机号
3. 手机号必须包含国家代码
4. 尝试部分匹配搜索

### Q: 优惠券发放失败？

**A**：
1. 检查优惠券是否在有效期内
2. 确认优惠券库存是否充足
3. 检查会员是否已经拥有该优惠券（部分优惠券限领一张）
4. 确认会员是否符合优惠券条件

### Q: 印花卡满了还能添加吗？

**A**：
- 如果印花卡已满，无法继续添加印花
- 会员需要先兑换奖励
- 或者创建新的印花卡活动

### Q: 如何查看会员的所有历史？

**A**：
1. 在会员详情页面点击"历史记录"
2. 可以按类型筛选（积分/优惠券/印花）
3. 可以按时间范围筛选
4. 可以导出历史记录到 Excel

### Q: 会员可以在多个品牌中使用吗？

**A**：
- 一个会员可以属于多个组（品牌）
- 每个组的积分、优惠券、印花是独立的
- 可以通过"添加到组"功能添加到其他品牌

- ✅ 记录修改原因

#### 添加积分
- ✅ 明确添加原因（如活动奖励、补偿）
- ✅ 检查每日/每周限制
- ✅ 设置合理的过期日期

#### 发放优惠券
- ✅ 确认优惠券有效性
- ✅ 检查会员是否符合条件
- ✅ 通知会员（通过短信/邮件）

### 数据安全

#### 权限管理
- ✅ 只有授权人员可以编辑会员信息
- ✅ 重要操作需要管理员权限
- ✅ 记录所有操作日志

#### 隐私保护
- ✅ 不要随意修改会员联系方式
- ✅ 修改密码后通知会员
- ✅ 避免在公开场合显示会员信息
