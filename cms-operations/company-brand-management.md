# CMS 公司品牌管理指南

## 标签
#user-guide #company #brand #shop #beginner

## 概述

本文档介绍如何管理 Storellet CMS 的组织架构，包括公司、组（品牌）和店铺的管理。

---

## 🏢 访问公司管理

### 步骤 1：进入公司管理页面

1. 登录 CMS 系统
2. 点击顶部导航的**"公司"**或**"Company"**
3. 系统会跳转到公司管理首页
   ```
   URL: /cms/company
   ```

### 步骤 2：查看公司列表

页面会显示：
- 公司总数
- 公司列表
- 排序选项

---

## 🏢 公司管理（Company）

### 什么是公司

公司（Company）是 CMS 组织架构的顶层，代表一个企业或组织。每个公司可以有多个组（Group/品牌）。

### 查看公司列表

显示所有公司：

| 字段 | 说明 |
|------|------|
| **公司 ID** | 公司唯一标识 |
| **公司名称** | 公司名称 |
| **加入日期** | 公司加入系统的日期 |
| **状态** | 启用/禁用 |
| **最后更新** | 最后更新时间 |
| **操作** | 编辑 |

### 排序功能

支持按以下字段排序：
- **ID**：按公司 ID 排序
- **名称**：按公司名称排序
- **加入日期**：按加入日期排序
- **状态**：按状态排序

**排序方式**：
- 升序（A→Z，1→10）
- 降序（Z→A，10→1）

### 创建公司

#### 步骤 1：点击新建

1. 在公司列表页面
2. 点击**"新建公司"**或**"Add"**
   ```
   URL: /cms/company/0 (or /cms/company/{id} with id <= 0)
   ```

#### 步骤 2：填写公司信息

**基本信息**：
- **公司名称**：输入公司的全称
- **加入日期**：选择公司加入系统的日期
- **状态**：
  - `1` = 启用
  - `0` = 禁用
- **创建日期**：默认当前时间

#### 步骤 3：保存公司

1. 点击**"保存"**或**"Submit"** (operation=submit)
   ```
   URL: /cms/company/{id} (POST with opt=submit)
   ```
2. 系统会创建公司

### 编辑公司

1. 在公司列表中找到要编辑的公司
2. 点击公司 ID 进入编辑页面
   ```
   URL: /cms/company/{id}
   ```
3. 修改公司信息
4. 点击**"保存"**

**注意事项**：
- ✅ 可以修改公司名称
- ✅ 可以更改公司状态
- ⚠️ 禁用公司会影响所有关联的品牌和店铺

## 🏷️ 组/品牌管理（Group/Brand）

### 什么是组/品牌

组（Group）对应品牌（Brand），一个公司可以有多个品牌。每个品牌代表一个独立的业务单元，拥有独立的：
- 用户数据
- 积分系统
- 优惠券系统
- 印花卡系统

### 访问组管理

1. 在公司管理页面
2. 点击**"组"**或**"Group"**
3. 系统会显示所有组
   ```
   URL: /cms/company/group
   ```

### 查看组列表

显示所有组：

| 字段 | 说明 |
|------|------|
| **组 ID** | 组唯一标识 |
| **组名称** | 组/品牌名称 |
| **公司** | 所属公司 |
| **积分比例** | 积分兑换比例 |
| **会员数量** | 该组的会员总数 |
| **状态** | 启用/禁用 |
| **操作** | 编辑 |

### 搜索组

**关键词搜索**：
1. 在搜索框输入关键词
2. 系统会搜索：
   - 组名称
   - 组 ID
3. 显示匹配结果

### 排序功能

支持按以下字段排序：
- **ID**：按组 ID 排序
- **名称**：按组名称排序
- **积分比例**：按积分比例排序
- **公司**：按所属公司排序
- **会员数量**：按会员数量排序
- **状态**：按状态排序

### 创建组

#### 步骤 1：点击新建

1. 在组列表页面
2. 点击**"新建组"**或**"Add"**
   ```
   URL: /cms/company/group/0
   ```

#### 步骤 2：填写组信息

**基本信息**：
- **组名称**：输入组/品牌的名称
- **公司**：选择所属公司
- **积分比例**：设置积分兑换比例（例如：100 积分 = 1 元）
- **状态**：
  - `1` = 启用
  - `0` = 禁用

**高级设置**：
- **货币**：选择组使用的货币
- **语言**：选择默认语言
- **时区**：选择时区

#### 步骤 3：保存组

1. 点击**"保存"**
   ```
   URL: /cms/company/group/{id} (POST with opt=submit)
   ```
2. 系统会创建组

### 编辑组

1. 在组列表中找到要编辑的组
2. 点击组 ID 进入编辑页面
   ```
   URL: /cms/company/group/{id}
   ```
3. 修改组信息
4. 点击**"保存"**

**可修改的信息**：
- ✅ 组名称
- ✅ 积分比例
- ✅ 状态
- ⚠️ 修改积分比例会影响积分计算

### 组的状态管理

#### 启用组

1. 打开组编辑页面
2. 将状态设为 **1**（启用）
3. 点击**"保存"**

**启用后**：
- ✅ 用户可以注册
- ✅ 可以正常使用积分和优惠券
- ✅ 可以查看报表

#### 禁用组

1. 打开组编辑页面
2. 将状态设为 **0**（禁用）
3. 点击**"保存"**

**禁用后**：
- ⚠️ 新用户无法注册
- ⚠️ 积分和优惠券功能暂停
- ⚠️ 用户无法查看信息
- ✅ 历史数据保留

## 🏪 店铺管理（Shop）

### 什么是店铺

店铺（Shop）属于某个品牌，代表具体的实体店或线上店铺。每个店铺有独立的：
- 店铺设置
- 店铺员工
- 交易数据

### 访问店铺管理

1. 在公司管理页面
2. 点击**"店铺"**或**"Shop"**
3. 系统会显示所有店铺
   ```
   URL: /cms/company/shop
   ```

### 查看店铺列表

显示所有店铺：

| 字段 | 说明 |
|------|------|
| **店铺 ID** | 店铺唯一标识 |
| **店铺名称** | 店铺名称 |
| **品牌** | 所属品牌 |
| **店铺类型** | 实体店/线上店 |
| **地址** | 店铺地址 |
| **电话** | 联系电话 |
| **状态** | 启用/禁用 |
| **操作** | 编辑 |

### 创建店铺

#### 步骤 1：点击新建

1. 在店铺列表页面
2. 点击**"新建店铺"**
   ```
   URL: /cms/company/shop/0
   ```

#### 步骤 2：填写店铺信息

**基本信息**：
- **店铺名称**：输入店铺名称
- **品牌**：选择所属品牌
- **店铺类型**：
  - 实体店
  - 线上店
  - 两者都是

**联系信息**：
- **地址**：输入店铺地址
- **电话**：输入联系电话
- **邮箱**：输入联系邮箱（可选）

**营业时间**（可选）：
- **营业开始时间**：设置每日营业开始时间
- **营业结束时间**：设置每日营业结束时间
- **休息日**：选择店铺不营业的日期

**状态**：
- **1** = 启用
- **0** = 禁用

#### 步骤 3：保存店铺

1. 点击**"保存"**
   ```
   URL: /cms/company/shop/{id} (POST)
   ```
2. 系统会创建店铺

### 编辑店铺

1. 在店铺列表中找到要编辑的店铺
2. 点击店铺 ID 进入编辑页面
   ```
   URL: /cms/company/shop/{id}
   ```
3. 修改店铺信息
4. 点击**"保存"**

## 🌟 特色项目（Featured Items）

### 什么是特色项目

特色项目用于在应用首页展示推荐的促销活动、品牌或店铺。

### 访问特色项目

1. 在公司管理页面
2. 点击**"特色项目"**或**"Featured Item"**
3. 系统会显示所有特色项目
   ```
   URL: /cms/company/featuredItem
   ```

### 查看特色项目列表

显示所有特色项目：

| 字段 | 说明 |
|------|------|
| **项目 ID** | 项目唯一标识 |
| **活动名称** | 特色项目名称 |
| **图片** | 项目封面图 |
| **链接** | 跳转链接 |
| **顺序** | 显示顺序 |
| **状态** | 启用/禁用 |
| **操作** | 编辑、删除 |

### 创建特色项目

#### 步骤 1：点击新建

1. 在特色项目列表页面
2. 点击**"新建特色项目"**

#### 步骤 2：填写项目信息

**基本信息**：
- **活动名称**：输入项目显示名称
- **链接**：输入跳转链接（例如：campaign://123）
- **顺序**：设置显示顺序（数字越小越靠前）
- **状态**：
  - `1` = 启用
  - `0` = 禁用

**图片上传**：
- **封面图**：上传项目封面图片
- 建议尺寸：1080x1080 像素
- 支持格式：JPEG、PNG
- 文件大小：不超过 5MB

#### 步骤 3：保存项目

1. 点击**"保存"**
   ```
   URL: /cms/company/featuredItem/{id} (POST)
   ```
2. 系统会创建特色项目

### 编辑特色项目

1. 在特色项目列表中找到要编辑的项目
2. 点击项目 ID 进入编辑页面
   ```
   URL: /cms/company/featuredItem/{id} (GET)
   ```
3. 修改项目信息
4. 点击**"保存"**

### 删除特色项目

1. 在特色项目列表中找到要删除的项目
2. 点击**"删除"**
3. 确认删除

## 🖼️ 品牌图片管理

### 什么是品牌图片

品牌图片用于品牌在应用中的展示，包括：
- Logo
- 封面图
- 促销图片

### 上传品牌图片

#### 步骤 1：进入图片管理

1. 在品牌编辑页面
2. 找到**"图片管理"**部分
3. 系统会显示已上传的图片

#### 步骤 2：上传图片

1. 点击**"选择图片"**
2. 选择图片文件
3. 输入图片描述（可选）
4. 点击**"上传"**
   ```
   URL: /cms/company/brand/{brandId}/{imageName}
   ```

#### 图片要求

**Logo**：
- 建议尺寸：512x512 像素
- 格式：PNG（支持透明背景）
- 大小：不超过 2MB

**封面图**：
- 建议尺寸：1080x1920 像素
- 格式：JPEG、PNG
- 大小：不超过 5MB

### 删除品牌图片

1. 在图片列表中找到要删除的图片
2. 点击**"删除"**
   ```
   URL: /cms/company/brand/{brandId}/{imageName} (DELETE)
   ```
3. 确认删除

**注意事项**：
- ⚠️ 删除图片会影响正在使用该图片的地方
- ⚠️ 删除前确认没有其他配置引用该图片

## 🔗 组织架构关系

### 层级结构

CMS 的组织架构分为三层：

```
公司 (Company)
  └─ 组/品牌 (Group/Brand)
      └─ 店铺 (Shop)
```

### 示例

**公司**：ABC 零售集团

**组/品牌**：
- Brand A（服装品牌）
- Brand B（餐饮品牌）
- Brand C（美妆品牌）

**店铺**：
- Brand A 下：
  - Store A1（中环店）
  - Store A2（铜锣湾店）
- Brand B 下：
  - Store B1（尖沙咀店）

### 数据隔离

不同层级之间的数据隔离：

| 层级 | 隔离内容 |
|------|----------|
| **公司** | 完全独立，所有数据隔离 |
| **组/品牌** | 用户、积分、优惠券、印花卡独立 |
| **店铺** | 交易、员工、库存独立 |

## 🎯 最佳实践

### 组织架构设计

#### 公司设置

**原则**：
- ✅ 一个企业实体 = 一个公司
- ✅ 避免创建过多公司
- ✅ 公司用于区分不同企业

#### 组/品牌设置

**原则**：
- ✅ 一个业务品牌 = 一个组
- ✅ 品牌之间需要数据独立
- ✅ 不同会员体系使用不同组

**何时需要多个组**：
- 品牌完全独立
- 不同的会员体系
- 不同的积分规则
- 不同的营销活动

#### 店铺设置

**原则**：
- ✅ 一个实体店 = 一个店铺
- ✅ 线上店也可以创建为店铺
- ✅ 店铺用于区分交易和员工

### 命名规范

#### 公司命名

- ✅ 使用企业法定名称
- ✅ 例如："ABC Retail Limited"
- ❌ 避免使用缩写或代号

#### 组/品牌命名

- ✅ 使用品牌官方名称
- ✅ 例如："BrandA"、"BrandB Premium"
- ❌ 避免使用模糊名称

#### 店铺命名

- ✅ 包含品牌 + 地区
- ✅ 例如："BrandA Central"、"BrandB TST"
- ❌ 避免使用编号

### 状态管理

#### 启用/禁用原则

**公司级别**：
- ⚠️ 禁用公司会禁用所有下属品牌和店铺
- ✅ 仅在公司完全停止运营时禁用

**组/品牌级别**：
- ✅ 品牌不再运营时禁用
- ✅ 数据保留可查看历史

**店铺级别**：
- ✅ 店铺关闭时禁用
- ✅ 重新开张时重新启用

## 🔧 常见问题

### Q: 如何修改公司名称？

**A**：
1. 进入公司管理页面
2. 找到要修改的公司
3. 点击公司 ID 进入编辑页面
4. 修改公司名称
5. 点击**"保存"**

### Q: 如何添加新品牌？

**A**：
1. 进入组管理页面
2. 点击**"新建组"**
3. 填写品牌信息
4. 选择所属公司
5. 点击**"保存"**

### Q: 如何查看某个品牌的所有店铺？

**A**：
1. 进入店铺管理页面
2. 使用品牌筛选
3. 选择要查看的品牌
4. 系统会显示该品牌的所有店铺

### Q: 禁用品牌后数据会丢失吗？

**A**：
- ✅ 数据不会丢失
- ✅ 历史数据完整保留
- ✅ 可以随时重新启用
- ⚠️ 禁用期间新功能暂停

### Q: 如何更改品牌的积分比例？

**A**：
1. 进入组管理页面
2. 找到要修改的品牌
3. 点击品牌 ID 进入编辑页面
4. 修改积分比例
5. 点击**"保存"**

**注意事项**：
- ⚠️ 修改积分比例会影响积分计算
- ⚠️ 已发放的积分不会重新计算
- ⚠️ 修改后新生效

### Q: 可以为一个品牌创建多个店铺吗？

**A**：
- ✅ 可以
- ✅ 没有数量限制
- ✅ 每个店铺独立管理交易和员工

### Q: 如何删除店铺？

**A**：
1. 进入店铺编辑页面
2. 滚动到页面底部
3. 点击**"删除"**
4. 确认删除

**注意事项**：
- ⚠️ 删除操作不可逆
- ⚠️ 历史交易数据保留
- ⚠️ 店铺员工会被移除
